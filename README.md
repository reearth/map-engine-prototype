# map-engine-prototype

3D globe map engine prototype built on Rust + WebAssembly (WIP)

## Development

### Install toolchains

- Rust (stable)
- Node.js (LTS)

### Install prerequisites

```console
cargo install cargo-make
cargo install cargo-watch
```

### Run with hot-reload

```console
cargo make dev
```

```console
open http://localhost:5174/
```
